## 当前Grafana版本

## 使用教程

**使用前请确认Grafana版本是否一致，Grafana版本更新方法请参考官网教程。**

复制`public`目录到Grafana资源目录(一般在`/usr/share/grafana`，建议先备份原有文件)，替换完成后重启Grafana服务即可。


## 测试教程

运行test文件夹下grafana-server.exe启动服务器，浏览器打开[http://localhost:3000][3]即可查看汉化版本内容。
