## 使用教程

复制`public`目录到grafana安装目录替换（替换！不可覆盖文件，建议将原目录重命名备份），替换完成后重启grafana即可。


## 测试教程

运行test文件夹下对应版本grafanaServer启动服务器，浏览器打开[http://localhost:3000][2]即可查看汉化版本内容。
